# 工作项目文档站

这个站点会把 `D:\obsidian\工作` 下的每个一级目录视作一个独立项目，并自动整理成适合在线阅读的 wiki 页面。

> 你可以把这里理解成“工作文档总入口”：先看项目总览，再进入每个项目主页，最后阅读项目里的原始文档和补充页。

## 当前站点结构

- `首页`：介绍整个工作文档站的用途和入口。
- `工作项目总览`：自动列出 `D:\obsidian\工作` 下的全部一级项目目录。
- `项目主页`：每个项目都有自己的主页，展示自动同步的 `文档/` 内容和项目补充页。
- `站点维护`：记录同步、构建、发布和排查方法。

## 使用方式

1. 先进入 [工作项目总览](./项目/index.md)。
2. 选择你想查看的项目，例如 `一句话编程` 或 `导盲兔`。
3. 在项目主页里进入 `文档/` 目录，或先看补充页了解项目结构。
4. 如果文档在 Obsidian 里更新，只要重新执行同步脚本或使用插件一键发布即可。
